Vue 3 基礎サンプル

宣言的レンダリング（Hello Vue）とリアクティビティ（カウンター）

1. 宣言的レンダリング（Hello Vue）

テンプレート内の {{'{{ message }}'}} とデータを、Vue が自動的に同期します。

{{ message }}
HTML/JSのコードを見る 
<div id="hello-vue">
  {{'{{ message }}'}}
</div>

const HelloVue = {
  data() {
    return { message: 'Hello Vue' }
  }
}
Vue.createApp(HelloVue).mount('#hello-vue')

2. リアクティビティ（カウンター）

@click でデータ count を更新すると、ボタン表示が自動で変わります。

HTML/JSのコードを見る 
<div id="counter-app">
  <button @click="count++">Count is: {{'{{ count }}'}}</button>
</div>

const CounterApp = {
  data() {
    return { count: 0 }
  }
}
Vue.createApp(CounterApp).mount('#counter-app')

使い方

  1. このHTMLファイルを保存（例：index.html）。
  2. ブラウザで開くと、そのまま動作します（CDNでVue 3を読み込み）。
